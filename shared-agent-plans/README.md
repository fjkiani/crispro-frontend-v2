# Shared Agent Plans Directory

This directory contains plan documents that are accessible to all agents.

**Location**: `~/Desktop/shared-agent-plans/`

**Purpose**: 
- Plans accessible outside git repos
- Shared between agents (Manager, Zo2, etc.)
- Not tracked in git

**Files**:
- `FINAL_COMPREHENSIVE_PLAN_ALIGNED.md` - Main implementation plan
- `REALITY_CHECK_COMPLETE_ASSESSMENT.md` - Current state assessment
- `mechanism-trial-matching.md` - Mechanism-based trial matching plan
- `zo2.mdc` - Zo2's SPE/WIWFM outcome prediction plan

**Usage**: 
- Agents can read/write to this directory
- Files persist across sessions
- Not version controlled (intentionally)
