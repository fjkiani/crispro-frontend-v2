CRISPR Terms
Term	Definition	When to Use
sgRNA	Single guide RNA - directs Cas9 to target	"Our AI designs sgRNAs with 80% efficacy"
PAM sequence	Protospacer adjacent motif - required for Cas9 binding	"We optimize sgRNAs near PAM sites"
On-target	Desired cutting location	"We achieve high on-target specificity"
Off-target	Unintended cutting locations	"We minimize off-target effects using chromatin data"
Indels	Insertions/deletions from NHEJ repair	"Our guides produce frameshift indels"
Genomics Terms
Term	Definition	When to Use
Chromatin accessibility	How "open" DNA is for proteins to bind	"We model chromatin accessibility to predict guide efficacy"
Enhancers	Regulatory elements that increase gene expression	"We identify enhancers regulating metastasis genes"
TADs	Topologically associating domains - 3D genome structure	"We account for TAD boundaries in our predictions"
Noncoding genome	The 99% of genome that doesn't code for proteins	"We analyze the noncoding genome, not just exons"
Cancer Terms
Term	Definition	When to Use
Driver mutation	Mutation that causes cancer	"We distinguish drivers from passengers"
Metastatic cascade	8-step process of cancer spread	"We target genes in the metastatic cascade"
Tumor microenvironment	Non-cancer cells surrounding tumor	"We consider TME interactions"
Therapeutic window	Dose range between efficacy and toxicity	"Our specificity increases the therapeutic window"
AI Terms
Term	Definition	When to Use
Foundation model	Large model pretrained on massive dataset	"We use Evo 2, a genomic foundation model"
Transfer learning	Fine-tuning pretrained model on specific task	"We transfer learn Evo 2 on cancer data"
Attention mechanism	How transformers weigh input importance	"Evo 2's attention learns long-range dependencies"
SAE (Sparse Autoencoder)	Method for extracting interpretable features	"Our 32,768 SAE features are biologically meaningful"
